author
Primary author of the document.
copyright
Name (or names) of copyright holder(s) for the document, and/or a statement of copyright
created
Date the document was created in ISO 8601 format.
i.e. YYYY-MM-DD
description
Brief description of the nature of the document.
domain
Root domain that the document belongs to.
filename
Outward facing name of the document.
generator
The application used to generate the document.
keywords
Comma and/or space seperated list of keywords within the document
modified
Date the document was last modified in ISO 8601 format.
i.e. YYYY-MM-DD
